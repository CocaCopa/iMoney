# iMoney!

iMoney! is a personal finance application developed in Unity that focuses on applying and refining clean, modular structures while gaining deeper experience with Android build pipelines and APK workflows.

## Purpose
- Further develop clean architecture practices  
- Explore modular UI flows and state management  
- Strengthen understanding of mobile deployment and build processes  
- Establish a simple, modern budgeting tool as the final outcome

## Scope
The goal of iMoney! is not scale, but clarity and maintainability.  
The app prioritizes:
- Straightforward functionality  
- Modern, lightweight UI  
- A codebase structured for readability and long-term extension

## Notes
This repository reflects ongoing work, experimentation, and incremental refinement of the underlying architecture.
